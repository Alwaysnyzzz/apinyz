Multi Upload
Upload maksimal 5 file sekaligus. Response berisi array URL publik untuk semua file.
Base Information
Base URL
https://api.nyzz.my.id
Endpoint
/upload/multiple
Method
POST
Content-Type
multipart/form-data
Max Files
5 file
Max Size
10 MB/file
Form Parameters
ParameterTypeRequiredKeterangan
files file[] Required Array file yang diupload. Gunakan field name files (bukan file)
Contoh Request
cURL
curl -X POST https://api.nyzz.my.id/upload/multiple \
  -F "files=@foto1.jpg" \
  -F "files=@foto2.png" \
  -F "files=@foto3.gif"
JavaScript / Fetch
const form = new FormData();
// Tambahkan semua file dengan nama field "files"
for (const file of fileInput.files) {
  form.append('files', file);
}

const res = await fetch('https://api.nyzz.my.id/upload/multiple', {
  method: 'POST',
  body: form
});
const data = await res.json();
// data.data = array of { filename, originalname, size, link }
data.data.forEach(f => console.log(f.link));
