Image Upload
Upload satu file dengan form-data. Response berisi URL publik file yang sudah diupload.
Base Information
Base URL
https://api.nyzz.my.id
Endpoint
/upload
Method
POST
Content-Type
multipart/form-data
Max Size
10 MB
Auth
Tidak perlu
Form Parameters
ParameterTypeRequiredKeterangan
file file Required File yang diupload. Format: jpg, jpeg, png, gif, webp, mp4, pdf, zip, txt
Contoh Request
cURL
curl -X POST https://api.nyzz.my.id/upload \
  -F "file=@gambar.jpg"
JavaScript / Fetch
const form = new FormData();
form.append('file', fileInput.files[0]);

const res = await fetch('https://api.nyzz.my.id/upload', {
  method: 'POST',
  body: form
});
const data = await res.json();
console.log(data.data.link); // URL file
